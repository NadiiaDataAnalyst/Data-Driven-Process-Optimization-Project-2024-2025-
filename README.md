# Data-Driven-Process-Optimization-Project-2024-2025-
Action: Extracted, cleaned, and aggregated historical data on customs declarations filed between 2024 and 2025 using Python (Pandas).  
Analyzed individual broker performance, workload distribution, and seasonal processing bottlenecks. Built interactive dashboards (Power BI) to visualize declaration dynamics and broker efficiency.
